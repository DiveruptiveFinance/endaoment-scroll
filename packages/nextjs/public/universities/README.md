# University Logos

Place university logos here with the following naming convention:

- `unam.png` or `unam.svg` - UNAM
- `ibero.png` or `ibero.svg` - IBERO
- `buap.png` or `buap.svg` - BUAP
- `udlap.png` or `udlap.svg` - UDLAP
- `anahuac.png` or `anahuac.svg` - ANAHUAC
- `tec.png` or `tec.svg` - TEC de Monterrey

## Order (as displayed in UI):
1. UNAM
2. IBERO
3. BUAP
4. UDLAP
5. ANAHUAC
6. TEC de Monterrey

## Usage in Code:
```typescript
const universityLogos = {
  unam: "/universities/unam.png",
  ibero: "/universities/ibero.png",
  buap: "/universities/buap.png",
  udlap: "/universities/udlap.png",
  anahuac: "/universities/anahuac.png",
  tec: "/universities/tec.png",
};
```
